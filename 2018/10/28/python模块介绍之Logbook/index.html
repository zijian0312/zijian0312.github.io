Oct 28, 2018

Logbook 是python中用于替代标准库logging的日志系统。

Logbook定义了很多Handler，如：

  • StreamHandler: 以流的形式记录日志
  • StderrHandler: 将日志记录到标准输出
  • FileHandler, MonitoringFileHandler, RotatingFileHandler and TimedRotatingFileHandler: 这些都是将日记记录到文件中

另外，还可以使用with指定日志系统的作用域

否则，日志系统将作用于整个文件。

Logbook日志级别

级别 说明
critical 严重错误，需要退出程序
error 错误，但在可控范围内
warning 警告
notice 大多数情况下希望看到的记录
info 大多数情况不希望看到的记录
debug 调试程序时详细输出的记录

Logbook应用

新建log.py文件，指定日志类型，同时输出到控制台和文件。
