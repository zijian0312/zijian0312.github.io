linux安装Python3.x

Sep 17, 2018

由于linux系统里有的程序需要依赖Python2，所以安装Python3.x的前提是不改变Python2的环境。

安装Python依赖环境

安装Python3.x

  1. 创建安装目录/usr/local/python3

  2. 下载Python3.5.2(版本号可直接在地址上修改)

  3. 解压安装文件

  4. 进入解压后的目录，编译安装


    如果安装出错，可能是缺少C++编译器。
    yum install gcc用于安装gcc。

  5. 建立软链接。

  6. 测试
    查看Python和pip版本
