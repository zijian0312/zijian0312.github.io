Markdown语法

May 18, 2018

常用功能

标题

[h1-h6] 标题

注： # 后面必须有空格
在hexo的apollo主题中，标题只支持h1和h4

上下文标题

注：= - 最少可以只写一个，兼容性一般

强调

斜体

加粗

删除线

加粗+斜体

语义标签

演示
 斜体 加粗 强调 Za Za Ctrl

代码

使用代码块要与上行距离一空行

  1. 代码块标记(´´´)

  2. 代码块缩进表示法

  3. 语法高亮显示(´´´+编程语言)
    apollo主题支持的编程语言有:
    HTML、 JS、 BASH、 CSS、 SCSS、 JAVA、 XML、 PYTHON、 JSON、
    SWIFT、 RUBY、 PERL、 PHP、 C、 CPP、 TS、 C++

  4. 内联代码块

表格

简约写法

演示

a b c
居中 左对齐 右对齐
========= =============== ============

:代表对齐方式，：和|之间不要有空格。
markdown不支持单元格的合并。

链接

内链式

百度   [百度](http://www.baidu.com/)

[百度](http://www.baidu.com/){:target="_blank"}

{:target="_blank"}跳转方式兼容性不好，多数平台都不支持（包括本博客）
