文件夹中所有文件名称的修改及输出

Jun 9, 2016

使用背景

我有一个文件夹，里面有几十首歌曲，我想给每首歌曲的名称前加个数字如：
  001_邓紫棋-喜欢你.flac
  002_GALA-追梦赤子心.flac
  ……
然后再输出到一个txt文档内（或者Excel）。

方法步骤

修改文件名称

  1. 下载一个小软件Replace Pioneer

  2. 安装完成后打开软件，找到 Tools -> Batch Runner

  3. Pick Files 选择所要修改的文件内容，确定后所有要修改的文件都会显示在下面的表格中。

  4. 勾选Set output file name，然后在后面的复选框中进行选择要修改的样式。如：file => 001_filename”，就表示在文件名称前加上数字，并依次递增。

输出文件名称

要将该文件夹内所有歌曲名称以列表形式显示到txt文本文档中。只需要用到 DOS 命令。

I 为盘符名称
folder 为文件夹名称
/b 表示只显示文件的名称，不显示创建日期和文件大小
list.txt 为生成的列表文件，如果要生成excel文件，可直接修改后缀为.xls
